## Cover images

Each shop card also shows a banner photo behind the logo. Drop these here, named
to match the `coverImage` path in `lib/shops.ts`:

- bebe-tailor-cover.jpg
- tuong-tailor-cover.jpg
- bao-diep-tailor-cover.jpg
- blue-xanh-tailor-cover.jpg

A wide/landscape photo (roughly 3:1 or wider) works best since the banner is cropped
to a short strip. If you upload a different file type (e.g. `.png` or `.webp`), update
the `coverImage` extension for that shop in `lib/shops.ts` to match. Until a cover image
is added, the card shows a plain gold-tint banner instead.
